# 📝 Список задач — React SPA

Учебное React-приложение для управления списком задач с маршрутизацией и загрузкой данных.

## 🚀 Онлайн-версия

**[Открыть приложение](https://KankaDesign.github.io/react-todo-app)**

## 🛠 Технологии

- React 18
- React Router 6
- Vite
- CSS (анимации, розовая цветовая гамма)

## 📦 API

Используется эмуляция API через локальный файл `data.js` (структура как у JSONPlaceholder).

## 🔧 Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка production-версии
npm run build

# Предпросмотр сборки
npm run preview

# 📱 Функционал

- ✅ Загрузка задач
- ✅ Добавление новых задач
- ✅ Отметка о выполнении (чекбокс)
- ✅ Удаление задач
- ✅ Переход между страницами
